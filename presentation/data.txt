Viola–Jones object detection - Признаки, используемые алгоритмом, опираются на суммирование пикселей из прямоугольных регионов. Учитывая изображение в оттенках серого, алгоритм анализирует множество окон разных размеров и положений и пытается обнаружить целевой объект, ища определенные особенности изображения в каждом окне. Несмотря на меньшую точность, чем современные методы обнаружения лиц, основанные на сверточных нейронных сетях (CNN), алгоритм Виолы-Джонса по-прежнему является эффективным решением для устройств с ограниченными ресурсами.

HOG Detector - HOG разлагает изображение на небольшие квадратные ячейки, вычисляет гистограмму ориентированных градиентов в каждой ячейке, нормализует результат, используя блочный шаблон, и возвращает дескриптор для каждой ячейки.

Deformable Parts Model (DPM) - Модель деформируемых деталей (DPM) (Felzenszwalb et al., 2010) распознает объекты со смешанной графической моделью (случайные марковские поля) деформируемых деталей. Модель состоит из трех основных компонентов:
- Грубый корневой фильтр определяет окно обнаружения, которое примерно покрывает весь объект. Фильтр определяет веса для вектора объектов региона.
- Фильтры из нескольких частей, которые охватывают меньшие части объекта. Фильтры частей обучаются с удвоенным разрешением корневого фильтра.
     Пространственная модель для оценки расположения фильтров частей относительно корня.

Region-Based Convolutional Neural Network : R-CNN или RCNN означает региональную сверточную нейронную сеть. Это тип модели машинного обучения, которая используется для задач компьютерного зрения, в частности для обнаружения объектов. Чтобы понять, что такое RCNN, мы рассмотрим архитектуру RCNN. Как работает R-CNN? На следующем изображении показана концепция CNN на основе региона (R-CNN). В этом подходе используются ограничивающие рамки между областями объекта, которые затем оценивают сверточные сети независимо от всех областей интереса (ROI) для классификации нескольких областей изображения в предлагаемый класс.
